# LocalPC - Local Productivity Command Center

A local-first command center for managing long text-based conversations, notes, and projects. 100% offline, no AI, no APIs.

## Features

- **Project System**: Organize work into local folders.
- **Conversation Viewer**: Import and navigate long documents.
- **Table of Contents**: Auto-generated from Markdown headings.
- **Offline Search**: Fast full-text search over local files.
- **Snapshots**: Save versioned snapshots of your work.

## Tech Stack

- **Frontend**: Vite, React, Tailwind CSS, Shadcn UI.
- **Backend**: Node.js (Express).
- **Storage**: Local JSON and Markdown files.

## Development

```bash
# Frontend
cd gpt/textgpt
npm install
npm run dev

# Backend
cd gpt
npm install
node server.js
```
